Assumptions:
	- For the "em" length unit a font size of 10 is assumed.
	- For the "ex" length unit ~ 0.5 * em is assumed, since we don't know 
	  neither the type of font nor the real size. 
	- If no width or height in an <svg> element is specified, the application
	  defaults to 500x500 pixels
	- The "none" argument for colour is relaised through a null value
